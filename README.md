# Daily Debug

**10秒日記アプリ - Googleスプレッドシート連携版**

---

## 🎯 コンセプト

毎日のコアプログレスを10秒で記録する、サイバーパンク風の日記アプリケーション。

---

## ✨ 機能

- 📊 **0-100のスコアスライダー** - 進捗を直感的に記録
- 📝 **複数行メモ入力** - 二言以内の短いメモ
- 📈 **Chart.jsによる成長グラフ** - 視覚的に進捗を確認
- ☁️ **Googleスプレッドシート同期** - クラウドでデータ管理
- 💾 **ローカルストレージ対応** - オフラインでも動作
- 📱 **PWA対応** - スマホにアプリとしてインストール可能
- 🎨 **レスポンシブデザイン** - スマホ・タブレット・PC対応
- 🌟 **ガラスモーフィズムUI** - サイバーパンク/フューチャリスティックな雰囲気

## 📊 Googleスプレッドシート連携セットアップ

データをクラウドで管理したい場合は、以下の手順でGoogleスプレッドシートと連携できます。

### 所要時間: 約10分

#### 手順：

1. **Googleスプレッドシートを作成**
2. **Google Apps Scriptをセットアップ**
   - `gas-script.gs` のコードをコピー&ペースト
3. **Web Appとしてデプロイ**
4. **script.jsにURLを設定**
5. **完成！**

📚 **詳細な手順は [SPREADSHEET_SETUP.md](./SPREADSHEET_SETUP.md) を参照してください**

## 💡 使い方

### 基本的な使い方

1. **スコアを設定**
   - スライダーを動かして、その日の進捗度（0-100）を設定

2. **メモを入力**
   - テキストエリアに短いメモを入力（二言以内がおすすめ）

3. **保存**
   - 「SEND TO CLOUD」ボタンをクリック

4. **自動更新**
   - グラフに自動的に反映され、データが保存されます

### 履歴を確認

- 右上のハンバーガーメニュー（≡）→「HISTORY (LOG)」
- 過去の記録が新しい順に表示されます

## 🆘 トラブルシューティング

### Q: データが表示されない
**A:** ブラウザをハードリフレッシュしてください（Ctrl+Shift+R / Cmd+Shift+R）

### Q: スプレッドシート連携が失敗する
**A:** `SPREADSHEET_SETUP.md` のトラブルシューティングセクションを参照してください

### Q: グラフが表示されない
**A:** Chart.jsのCDNが読み込まれているか確認してください（インターネット接続が必要）
