## 📱 基本的な使い方

### 1. アプリを開く

ブラウザで `index.html` を開くだけで使えます！

```bash
# 方法1: ファイルをダブルクリック
# 方法2: ブラウザにドラッグ&ドロップ
# 方法3: ターミナルから
open index.html
```

### 2. データを記録する

1. **スライダー** を動かして、今日の進捗度（0-100）を設定
2. **テキストエリア** に短いメモを入力
3. **「SEND TO CLOUD」** ボタンをクリック
4. データが保存され、グラフが更新されます！

### 3. 履歴を見る

1. 右上の **ハンバーガーメニュー** （≡）をクリック
2. **「HISTORY (LOG)」** を選択
3. 過去の記録が新しい順に表示されます

## 🚀 次のステップ（オプション）

### Googleスプレッドシート連携を追加する

現在、データはローカルにのみ保存されています。Googleスプレッドシートと連携すると：

✅ **どのデバイスからでもアクセス可能**
✅ **自動バックアップ**
✅ **データ分析がより簡単に**
✅ **複数デバイスで同期**

#### 簡単な手順（約10分）：

1. **Googleスプレッドシートを作成**
   - Google Driveで新しいスプレッドシートを作成

2. **Google Apps Scriptをセットアップ**
   - `gas-script.gs` のコードをコピー
   - スプレッドシートの「拡張機能」→「Apps Script」に貼り付け

3. **Web Appとしてデプロイ**
   - 「デプロイ」→「新しいデプロイ」
   - 「ウェブアプリ」を選択
   - URLをコピー

4. **script.jsにURLを設定**
   - 3行目の `GAS_URL` にコピーしたURLを貼り付け

5. **完成！**
   - データが自動的にスプレッドシートに保存されます

📚 **詳細な手順は `SPREADSHEET_SETUP.md` を参照してください**

## 🔧 トラブルシューティング

### Q: データが消えた
A: ブラウザのキャッシュをクリアすると消えます。重要なデータはスプレッドシート連携をセットアップしてバックアップしてください。

### Q: グラフが表示されない
A: Chart.jsのCDNが読み込まれているか確認してください。インターネット接続が必要です。

### Q: スマホで見にくい
A: **PWAとしてインストール** すると、フルスクリーンで使えます！
- **iPhone**: Safari → 共有ボタン → 「ホーム画面に追加」
- **Android**: Chrome → メニュー → 「ホーム画面に追加」

### Q: スプレッドシート連携がうまくいかない
A: まずローカル版で使い続けられます。連携は完全にオプションです。詳細は `SPREADSHEET_SETUP.md` を参照してください。
